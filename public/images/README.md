# Image Assets — Mil Jayega

Replace these placeholder references in src/data/images.js with your actual product photos.

## Recommended image sizes:
- Hero: 1600 x 900px (landscape)
- Product cards: 600 x 720px (portrait 5:6 ratio)
- Category tiles: 800 x 533px (landscape 3:2 ratio)
- About hero: 1200 x 800px

## Free image sources:
- https://unsplash.com (search: streetwear, tshirt, cargo pants)
- https://pexels.com
- https://pixabay.com

## How to add your images:
1. Put your images in this folder (public/images/)
2. Reference them in src/data/images.js as '/images/your-file.jpg'
